Penjelasan Program Testing Mobile Apps E-Commerce :

1. Automated testing dilakukan menggunakan perangkat lunak Katalon. 
2. Aplikasi E-Commerce yang diuji menggunakan sistem operasi Android. File .apk terdapat pada folder APK pada project ini.
3. Tedapat beberapa test case:

	- TC Add Item untuk menguji ketika membeli 1 item di aplikasi E-Commerce kemudian melakukan checkout pada Shopping Cart dengan memasukkan data pembeli dan memastikan jumlah barang dan total harga adalah benar. 
	- TC Add Several Item untuk menguji ketika membeli beberapa item di aplikasi E-Commerce kemudian melakukan checkout pada Shopping Cart dengan memasukkan data pembeli dan memastikan jumlah barang dan total harga adalah benar. 
	- TC Detail Profile untuk menguji ketika pengguna ingin melihat profile dengan memilih button edit. 
	- TC Order History untuk menguji ketika pengguna ingin melihat barang yang telah dipesan.
	- TC Privacy Policy untuk menguji ketika pengguna ingin melihat policy dari aplikasi
	- TC Rate Us untuk menguji ketika pengguna memilih untuk memberikan rating terhadap aplikasi E-Commerce.
	- TC Share untuk menguji ketika pengguna ingin menyebarkan item ke parangkat lain yang sudah terintegrasi dengan aplikasi E-Commerce. Pada aplikasi ini menampilkan Bluetooth dan Message.
	- TC Category untuk menguji ketika pengguna ingin melihat item berdasarkan category pilihan yang tertera pada list category.
	- TC Delete Item from Shopping Cart untuk menguji ketika pengguna ingin menghapus item yang telah ditambahkan sebelumnya pada Shopping Cart.
	- TC Help untuk menguji ketika pengguna ingin melihat panduan yang tersedia pada list help pada aplikasi E-Commerce
	- TC Search untuk menguji ketika pengguna ingin melihat item berdasarkan text masukan yang diberikan pada kolom search pada aplikasi E-Commerce.
	
4. Terdapat 3 Test Suite untuk kumpulan test case yang sesuai dengan Navigator Bar yang ada pada aplikasi E-Commerce.
	
	- TS Add Navigation 
	- TS Category and Help Navigation
	- TS Profile Navigation 

5. Pengguna dapat menjalankan Automated Testing Program ini dengan menekan button Play dan memilih jalankan menggunakan Android Device (device yang digunakan dapat menggunakan perangkat asli yang dihubungkan ke komputer maupun perangkat virtual yang diinstall di komputer pengguna).
6. Arahkan pengujian menggunakan aplikasi E-Commerce (file .apk yang telah disediakan dalam folder APK)
